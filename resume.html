INTRODUCE

백엔드 개발자를 목표로 다양한 프로젝트와 협업 경험을 쌓아왔습니다. Java와 Spring Boot를 기반으로 웹 서비스 개발을 진행하며 MySQL, Redis, AWS(EC2, S3, RDS) 등을 활용해 서버와 데이터 처리 구조를 직접 구현해왔고, JavaScript와 HTTP 기반 통신을 통해 프론트엔드와 연동되는 서비스 흐름을 이해하고자 노력해왔습니다.


학부 과정에서는 컴퓨터공학 전공 동아리에서 알고리즘 스터디에 참여하며 문제 해결 능력을 꾸준히 키워왔고, 팀 프로젝트를 통해 협업 경험을 쌓았습니다. 한이음 공모전과 전공 랩실 프로젝트에 참여하며 다양한 역할의 팀원들과 함께 서비스를 개발했고, 이를 통해 단순한 기능 구현을 넘어 협업과 소통의 중요성을 깊이 이해하게 되었습니다.


특히 프로젝트를 진행하며 단순한 기능 구현을 넘어 사용자 요청이 시스템에 미치는 영향을 고려하는 개발을 지향하게 되었습니다. 서버 부하를 줄이기 위한 데이터 처리 방식과 API 구조를 고민하고, AI 모델과 서버를 연동하는 과정에서도 전체 시스템 흐름을 고려해 구조를 설계하며 서비스 관점에서 문제를 해결하고자 노력해왔습니다.


앞으로는 백엔드 개발자로서 기술적인 성장뿐만 아니라, 서비스 구조를 이해하고 문제를 구조적으로 해결할 수 있는 개발자가 되고자 합니다. 팀과 함께 협업하며 더 나은 서비스를 만들어가는 과정 속에서 지속적으로 학습하고, 실제 서비스 환경에서 가치를 만들어낼 수 있는 개발자로 성장해 나가고자 합니다.
